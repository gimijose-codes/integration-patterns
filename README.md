# Integration Patterns

Exploration of enterprise integration patterns focused on reliability, 
observability, and third-party system resilience.

## Patterns covered
- Webhook receiver with signature validation
- Retry logic with exponential backoff
- Idempotent event processing
- SCORM/xAPI completion event handling

## Tech
Ruby, REST APIs, event-driven architecture

## Background
Built as part of exploring the e-learning integration space — 
specifically LMS integrations, HRIS user provisioning via SCIM, 
and SSO federation patterns.
